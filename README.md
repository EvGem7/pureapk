# pureapk
Example of building an Android app without Gradle.

## Building
0. JDK must be installed.
1. [aapt2](https://developer.android.com/tools/aapt2) has to be on PATH. The repo provides aapt2 built for Linux so on Linux you can `export PATH="$PWD:$PATH"`. On any other OS you have to provide aapt2 yourself.
2. `./make.sh`
